# Vibeful Desktop Releases

Build and release artifacts for [Vibeful](https://vibeful.io) Desktop App.

Download the latest version from the [Releases](https://github.com/Jiansen/vibeful-releases/releases) page.

| Platform | File |
|----------|------|
| macOS Apple Silicon (M1+) | `Vibeful_*.dmg` (aarch64) |
| macOS Intel | `Vibeful_*.dmg` (x64) |
| Windows 64-bit | `Vibeful_*.exe` / `.msi` |
| Linux (Debian/Ubuntu) | `Vibeful_*.deb` / `.AppImage` |
